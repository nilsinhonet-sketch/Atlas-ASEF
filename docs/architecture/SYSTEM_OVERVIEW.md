# SYSTEM OVERVIEW

## Objetivo

Este documento descreve a visão geral da arquitetura do Atlas ASEF.

## Componentes

O Atlas ASEF é dividido em três grandes blocos:

### 1. Core

Responsável pela metodologia.

- Manifesto
- Templates
- Checklists
- Protocolos
- Documentação

---

### 2. Engine

Responsável pela coordenação entre agentes de IA.

- Continuidade
- Planejamento
- Sprint
- Auditoria
- Validação

---
